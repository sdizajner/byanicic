— About

Design is thinking made visible.

Web design has always attracted me as a balance between aesthetics, structure and functionality. I started by working with Adobe Photoshop and Dreamweaver, then continued developing my skills through Xara Designer, WordPress, Figma, HTML, CSS and JavaScript. Over time, I built a design process that does not begin with a ready-made template, but with a clear understanding of the brand, the target audience and the first impression a website needs to create.

I completed a one-year education program at ITAcademy, Department of Design & Multimedia, specializing in Web Design, where I earned certificates in web design, Adobe Photoshop and Illustrator, Figma, WordPress and SEO optimization. Today, I combine that knowledge with practical experience in creating modern, responsive and visually refined websites.

My work focuses on UI/UX design, WordPress development, SEO-friendly website structure, visual identity and clean front-end code. I actively follow current design standards, improve my workflow with AI tools and use vibe coding to support faster prototyping, idea testing and process optimization.

— Services

Six things I do
properly.

  1. 01

    Web Design

    Marketing sites, portfolios and editorial pages built around the brand — not a template. Visuals are prepared in Adobe Photoshop and Illustrator, layout in Figma. Most sites I design end up between five and twenty pages.

  2. 02

    UI / UX Design

    Product interfaces, dashboards and end-to-end flows. I lean on real data and short feedback loops with the engineering team — backed by a solid grasp of HTML, CSS and JavaScript so the design stays buildable in code.

  3. 03

    WordPress Design & Build

    Custom WordPress themes for clients who want to update content themselves without breaking the layout. Editor-friendly, fast, with clean front-end code (HTML / CSS / JS) and an SEO-friendly structure baked in from the start.

  4. 04

    Design Systems

    Token-driven Figma libraries handed off as code. I document the things that usually get forgotten — spacing scales, motion, edge cases.

  5. 05

    Brand Identity & SEO

    Logotypes, type systems and the small print collateral — cards, decks, signage — crafted in Illustrator and Photoshop. Paired with on-page SEO optimization: structure, meta data, Core Web Vitals and clean semantic HTML.

  6. 06

    Vibe Coding — Working with AI Tools

    Rapid prototyping and idea testing through AI tools — from concept to working prototype in a day. I use AI to speed up the workflow, generate variants, support copywriting and tidy code, while the final aesthetic and logic stay hand-tuned.
